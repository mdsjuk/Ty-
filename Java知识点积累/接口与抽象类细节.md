# 接口与抽象类总结对比

## 	接口

- 接口里面的成员:

```java
 //无构造函数 不能被实例化
 //接口不能继承抽象类与普通类   接口只能extends接口
interface inter{
    //变量默认 public static final 修饰
    public static  final int a=1;
    //抽象方法默认 public abstract修饰
    public  abstract void method_1();
    //静态内部类
    static  class innerclass{
    }
    // 普通内部类
    class intf{ }
    // static修饰的普通方法
    static void method_2(){
        System.out.println("普通方法2");
    }
    //default修饰的普通方法
    default void  method_3(){
        System.out.println("普通方法3");
    }
}
```

- 接口的继承和实现接口

```java

interface A {
    void method_1();
    void method_2();
}
//接口可以继承接口 而且是只能
interface B extends A {
   void method_3();
}
// 普通类要实现全部的接口方法
class normal   implements B  {
    @Override
    public void method_1() {
        System.out.println("接口A的方法");
    }
    @Override
    public void method_2() {
        System.out.println("接口A的方法");
    }
    @Override
    public void method_3() {
        System.out.println("接口B的方法");
    }
}
// 抽象类可以实现部分的接口方法
abstract class abclass implements B{
    @Override
    public void method_1() {
        System.out.println("选择性重写，不需要全部重写");
    }
    abstract void method_3();
}
//普通类实现全部抽象类的方法 包括抽象类的继承
class normal2 extends abclass{
    @Override
    public void method_2() {
        System.out.println("重写接口方法");
    }

    @Override
    public void method_3() {
        System.out.println("重写接口方法");
    }
    
     @Override
        void method_4() {
             System.out.println("实现抽象类的方法");
        }
}
 
```

## 抽象类

    }
    private void method_4(){

    protected void method_5(){

    }
    static class inner{}

 class A_1 {

      A_1(){
